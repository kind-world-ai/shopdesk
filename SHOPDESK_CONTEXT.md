# ShopDesk Project Context

## Brand
- Name: ShopDesk
- Tagline: Professional Tools. Simple Price.
- Domain: theshopdesk.in
- Colors: Navy #1A1A2E | Orange #E87722 | Teal #00897B
- Fonts: Plus Jakarta Sans (headings) + Inter (body)

## Products
- Salon CRM: ₹999 one-time, Google Sheets, 11 sheets, 250+ formulas
- Construction CRM: ₹999 one-time, Google Sheets, 11 sheets, 373 formulas
- ShopDesk Pro: ₹2,999 — PDF automation + WhatsApp alerts
- Done-For-You: ₹9,999 — custom configuration

## Target Market
- Primary: Chandigarh, Mohali, Panchkula, Punjab, Haryana
- Buyers: Salon owners, civil contractors, billing engineers, PMCs
- Price sensitivity: High — ₹999 positioned as "less than one day's labour"
- Language: English + Hindi code-switching natural in content

## SEO Requirements
- Every page: H1 with primary keyword, meta description, JSON-LD schema
- FAQ section on every long-form page (FAQPage schema)
- AEO: Question-first H2s, definition blocks, stat-backed claims
- Internal links: Every page links to /salon, /construction, /pricing

## Tech Stack
- Next.js 16 App Router
- TypeScript
- Tailwind CSS v4
- MDX for blog posts (via @next/mdx or contentlayer)
- JSON-LD structured data in layout/page level
